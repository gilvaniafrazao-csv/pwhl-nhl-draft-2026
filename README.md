# 🏒 NHL & PWHL 2026 Draft — De Onde Vêm as Futuras Estrelas do Hockey?

Primeiro projeto de análise de dados | First data analytics project

---

## Sobre

Análise dos picks do 1º round dos drafts da NHL e PWHL 2026, com foco em identificar quais clubes, universidades e países mais produzem atletas para o hockey profissional.

## Principais Achados

- 🏒 **92%** dos picks da PWHL vieram de universidades da NCAA
- 🇨🇦 O **Canadá** lidera a NHL com 47% dos picks do 1º round

## Ferramentas

- **Excel** — curadoria dos dados
- **Figma** — design do dashboard
- **Power BI** — visualização de dados
